# A CSS3 TextMate Bundle

## Features

* border-radius
* box-shadow
* Odd/Even Selectors
* Columns (WebKit only)
* Text Stroke (WebKit only)
* text-shadow
* opacity
* display-inline

## Usage

Double click on the .tmbundle file and it'll install into TextMate.

## Contributing
